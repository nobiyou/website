使用指南

如何使用

下载视频

1. 打开微信视频号中的视频详情页

2. 在视频下方操作栏中点击下载按钮

注意：如果没有看到下载按钮，请检查「更多」选项中是否有「下载视频」。下载功能仅在视频详情页可用。

长视频下载

对于较长的视频，软件提供了缓存进度显示功能：

1. 视频加载过程中会显示缓存进度

2. 缓存完成后会有明显提示，此时可以进行下载

提示：长视频需要完整缓存后才能下载，建议按顺序缓存（不要跳着点进度条）

常见问题

常见问题解答

启动软件后，打开微信视频号中的视频详情页，在视频下方操作栏中会自动添加下载按钮，点击即可下载。如果没有看到下载按钮，请检查"更多"选项中是否有"下载视频"。

对于较长的视频，需要完整缓存后才能下载。视频加载过程中会显示缓存进度，缓存完成后会有明显提示，此时才能进行下载。建议按顺序缓存，不要跳着点进度条。

软件支持多种分辨率和质量的视频下载，包括1024x576、1280x720以及原始1920x1080高清格式。每种分辨率还有不同的码率选择，可以根据自己的需求选择合适的格式。

如果服务启动后视频详情一直在加载，终端无日志，在终端中按一次Ctrl+C即可恢复正常。

遇到解密失败，停止下载的情况，请关闭所有视频页面和窗口，然后重新打开尝试下载。

开发指南

开发指南

环境要求

  • Go语言环境
  • 管理员权限（用于网络请求拦截）

运行方式

以管理员身份启动终端，然后执行：

go run main.go

打包发布

# 基本打包
go build -o wx_channel.exe

# 优化体积的打包
go build -ldflags="-s -w" -o wx_channel_mini.exe

# 使用upx压缩
upx --best wx_channel.exe

技术实现

本项目的核心实现基于以下技术：

  1. 1 网络请求拦截获取视频资源
  2. 2 自定义界面元素添加下载按钮
  3. 3 视频流处理和缓存管理
  4. 4 数据分析和导出功能
